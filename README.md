# SnackManager Frontend

Frontend do sistema SnackManager: uma plataforma administrativa de gerenciamento de lojas, produtos, estoque, pedidos e funcionários, desenvolvida com React, TypeScript e Tailwind CSS.

## Tecnologias
- React
- TypeScript
- Tailwind CSS
- Axios
- React Router

## Scripts
- `npm run dev`: inicia o servidor de desenvolvimento
- `npm run build`: cria a versão de produção
- `npm run lint`: verifica problemas de lint
